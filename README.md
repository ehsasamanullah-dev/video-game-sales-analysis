# Global Video Game Sales Analysis

## Project Overview
This project analyzes global video game sales to identify trends across genres, platforms, and years.

## Tools Used
- Tableau
- Excel

## Dashboard Features
- Global sales trend by year
- Top platforms by sales
- Genre performance comparison
- Sales distribution analysis

## Key Insights
- Action games generated the highest global sales.
- Global video game sales peaked around 2008.
- North America contributed the largest share of global revenue.
- Shooter and Sports genres consistently ranked among the top-selling categories.
